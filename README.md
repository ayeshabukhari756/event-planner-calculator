# Event Planner Cost Calculator

A responsive front-end web application that allows users to estimate event planning costs instantly.

## Features

- Event Type Selection
- Basic Decoration
- DJ / Sound System
- Photography
- Videography
- Flower Decoration
- Lighting
- Cake
- Catering Cost per Guest
- Coupon Discount
- Tax Calculation
- Live Order Summary
- Responsive Design

## Technologies Used

- HTML5
- CSS3
- JavaScript
